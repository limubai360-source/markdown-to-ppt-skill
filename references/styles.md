# PPT HTML 样式参考

本文件记录四种可选 HTML PPT 样式。生成英文 deck 时可使用英文风格标签；生成中文 deck 时使用中文标签。默认推荐样式一。

## 样式一：Linear / Notion

英文标签：Linear / Notion style

适合表达：结构清晰、现代、高效、产品化。

视觉要求：

- 整体简约、清晰、轻量，强调信息层级和可读性。
- 背景以白色、浅灰色、淡蓝灰色为主，避免深色大面积背景。
- 使用卡片式布局，每个信息模块放在圆角卡片中。
- 卡片需要有轻微阴影或浅色边框，视觉上干净、不厚重。
- 字体风格接近 Inter / Helvetica / PingFang SC，标题清晰，正文克制。
- 配色以黑、白、灰为主，使用少量蓝色或紫色作为强调色。
- 页面布局偏产品文档感，适合展示问题、方案、能力模块、产品价值。
- 图形元素使用简洁线框图标、浅色标签、模块卡片、轻量箭头。
- 每页信息密度适中，不要过度装饰，也不要过于空洞。

HTML 要求：

- 页面比例为 16:9。
- 每一页使用 `<section class="slide">`。
- 输出完整单文件 HTML，包含 CSS。
- 不依赖外部图片资源。
- 方便后续转换为 PPTX。

## 样式二：Apple Keynote 极简

英文标签：Apple Keynote minimal style

适合表达：高级、愿景感、未来感、聚焦。

视觉要求：

- 整体高级、克制、留白充足，强调“大标题 + 少量文字 + 强视觉焦点”。
- 每页只表达一个核心观点，避免信息过密。
- 背景以纯白、浅灰、黑色或深色渐变为主。
- 标题字号要大，具备发布会式视觉冲击力。
- 正文内容要非常简洁，使用短句，不堆叠大量说明。
- 页面中可以使用大面积留白、抽象渐变、柔和光影、简洁几何形状。
- 如果有图形表达，使用大图形、大模块、少量线条，避免复杂流程图。
- 配色以黑、白、灰为基础，可使用少量蓝色、银灰色或淡紫色作为强调。
- 整体感觉接近 Apple 发布会、Keynote 产品介绍页，而不是传统商务 PPT。

HTML 要求：

- 页面比例为 16:9。
- 每一页使用 `<section class="slide">`。
- 输出完整单文件 HTML，包含 CSS。
- 不依赖外部图片资源，可用 CSS 绘制渐变、光影和几何形状。
- 方便后续转换为 PPTX。

## 样式三：咨询公司风格

英文标签：Consulting firm style

适合表达：专业、可信、逻辑严密、适合决策。

视觉要求：

- 整体专业、严谨、结构化，强调逻辑清晰和信息组织。
- 每页标题采用“结论先行”的表达方式，而不是普通描述性标题。
- 页面适合展示分析框架、对比矩阵、流程图、时间线、分层架构、实施路径。
- 信息密度可以适中偏高，但必须保持清晰的分组和层级。
- 配色以深蓝、灰色、白色为主，可使用少量强调色突出关键结论。
- 使用规整的网格布局，模块对齐严格，边距统一。
- 图形元素包括矩阵、编号模块、阶段箭头、表格、分层框、关键指标卡片。
- 不要使用过多渐变、插画或装饰性元素。
- 版面应像麦肯锡、BCG、贝恩、德勤等咨询公司常见汇报材料。

HTML 要求：

- 页面比例为 16:9。
- 每一页使用 `<section class="slide">`。
- 输出完整单文件 HTML，包含 CSS。
- 不依赖外部图片资源，图表尽量用 HTML + CSS 绘制。
- 方便后续转换为 PPTX。

## 样式四：现代 SaaS 官网风格

英文标签：Modern SaaS website style

适合表达：智能、开放、平台化、增长、产品价值。

视觉要求：

- 整体轻盈、现代、科技感强，像一个高质量 SaaS 产品官网。
- 背景以白色、浅蓝、浅紫、淡灰渐变为主。
- 使用大圆角卡片、柔和阴影、渐变按钮、标签、数据卡片等元素。
- 适合展示产品能力、核心架构、平台模块、客户价值、增长数据。
- 页面可以使用轻量插画感的 CSS 图形，例如仪表盘、数据图、模块面板、悬浮卡片。
- 配色以科技蓝、淡紫、白色、浅灰为主，整体明亮但不花哨。
- 标题要简洁有力，正文短句化，模块说明清晰。
- 可以使用 CTA 按钮、能力卡片、图标模块、产品界面示意图等 SaaS 官网常见元素。
- 页面视觉要比传统商务 PPT 更年轻、更产品化、更有科技感。

HTML 要求：

- 页面比例为 16:9。
- 每一页使用 `<section class="slide">`。
- 输出完整单文件 HTML，包含 CSS。
- 不依赖外部图片资源，尽量用 HTML + CSS 绘制产品界面和图形。
- 方便后续转换为 PPTX。
